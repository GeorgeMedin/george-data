miniAPI de Días Hábiles en Tiempo Real

Mientras desarrollaba uno de mis dashboards en Power BI 📊, me enfrenté al reto de calcular los días hábiles del mes actual — excluyendo sábados, domingos y festivos en Colombia 🇨🇴. Inicialmente lo resolví con Python 🐍 usando la librería holidays, pero el desafío real era: ¿Cómo obtener esa información de forma dinámica dentro de Power BI Service? 🤔⚡ Así nació esta mini API 🌐 desarrollada con Google Apps Script, la cual expone un endpoint en tiempo real que devuelve un JSON como este: 

{
    "Dias_Habiles_Desde_Inicio_Mes_Hasta_Ayer": 0,
    "Dias_Habiles_Totales_Mes_Actual": 23
}
🔗 Probar API Ideal para conectar directamente con Power BI o cualquier herramienta que consuma datos en formato JSON. 🚀

Encuesta de Satisfacción con Respuestas automatizadas

Automaticé la gestión de encuestas de satisfacción para un hotel usando n8n y Python. ✅ Funcionamiento: El cliente llena una encuesta. Un script en Python analiza las respuestas. Si hay insatisfacción, se genera y envía automáticamente un correo personalizado. Los datos se almacenan en Excel para análisis posterior. 💡 Impacto: Mejora la atención al cliente con respuestas inmediatas. Organiza la retroalimentación para tomar decisiones. Ahorra tiempo al automatizar tareas repetitivas. Este flujo fue desarrollado como experimento personal. Actualmente, el formulario no se encuentra activo.

Validador de correo electronico usando metodo SMTP

Script que verifica si un correo electrónico existe sin enviar mensajes. 🔹 Consulta los registros MX del dominio. 🔹 Se conecta vía SMTP y simula el envío. 🔹 Interpreta la respuesta del servidor (250, 550, etc.). 💡 Útil para validar emails y depurar bases de datos.

Query de Informe de Antigüedades SAP B1

Consulta optimizada para SAP Business One que genera un informe claro y eficiente de antigüedad de saldos en cuentas por cobrar, segmentado por rangos de días de mora. 🔹 Mejora el rendimiento frente a los informes nativos 🔹 Facilita el análisis financiero y la gestión de cartera 🔹 Soluciona las limitaciones comunes del reporte estándar en SAP B1
